Features:

  • Automatically adds 'div.clear:both' element after floating elements
  • Works in any browser, there's no CSS hacks, it uses old "clear: both" (and a little JS), exactly the same way as if you did it by hand
  • Browser should have JavaScript enabled
  • Current implemetation depends on MooTools (it's trivial to port it to another libraries, jQuer for example, but I'm too lazy to do it, maybe someone else someday will do that)
