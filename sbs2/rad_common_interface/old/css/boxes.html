Features:

  • Clean and semantic HTML, visual appearance 100% separated from data
  • Fixed or flexible (fluid) layout
  • Can be nested in any order, place and level without any care
  • Works in any browser, there's no CSS hacks, it uses old "float: left" and "clear: both" (and a little JS), exactly the same way as if you did it by hand
  • Specify width of key elements only (using any mix of em/%/px), let the computer calculate rest
  • Simple to use (only one method with 3 arguments) and small JS size
  • Browser should have JavaScript enabled
  • Current implemetation depends on MooTools (it's trivial to port it to another libraries, jQuer for example, but I'm too lazy to do it, maybe someone else someday will do that)
